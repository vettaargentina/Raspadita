ℹ️ ¿Cómo funciona?
  • 1️⃣ Configurá tus premios en la pestaña Premios.
  • 2️⃣ Configurá la URL base de tu GitHub Pages en Configuración (opcional pero recomendado).
  • 3️⃣ Generá la cantidad de cartones que necesitás. Cada uno tiene un ID único y los premios en orden aleatorio.
  • 4️⃣ Copiá cada link y compartilo con tu cliente. Cada link funciona una única vez.
  • 5️⃣ El cliente raspa, ve su premio y te manda una captura de pantalla para canjear.
